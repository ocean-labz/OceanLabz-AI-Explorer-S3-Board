# 📘 Getting Started with OceanLabz AI Explorer S3 Board

Welcome! This guide will walk you through everything you need to go from unboxing to running your first sketch.

---

## What's in the Box

| Item | Notes |
|------|-------|
| OceanLabz AI Explorer S3 Board | Main board |
| USB Type-C Cable | For power and programming |
| Pin Headers (if included) | May require soldering |

> **You will need:** A computer with a USB port and the Arduino IDE installed.

---

## Learning Path

For beginners, follow this order:

```
Step 1: Install Arduino IDE          ← You are here
Step 2: Install ESP32 Board Package
Step 3: Install Libraries
Step 4: Connect the Board
Step 5: Run Blink (your first sketch)
Step 6: Run Serial Test
Step 7: Explore Display / Audio / Camera guides
```
