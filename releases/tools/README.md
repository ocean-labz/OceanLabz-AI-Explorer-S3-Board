# Release Tools

Scripts and configuration files bundled with each release for flashing and testing the AI Explorer S3 board.

## Contents

- `flash_tool/` — Espressif Flash Download Tool preset configurations
- `factory_firmware/` — Pre-compiled `.bin` files for factory testing
- `bootloader/` — Bootloader binaries (if customised)

> Refer to the main [`releases/README.md`](../README.md) for the release process.
