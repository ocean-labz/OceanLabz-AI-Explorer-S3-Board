# 🎨 Assets

Media assets for the OceanLabz AI Explorer S3 Board repository.

## Structure

```
assets/
├── images/          # Screenshots, photos, reference images
├── datasheets/      # Component datasheets (PDF)
├── diagrams/        # Block diagrams, pinout diagrams, wiring diagrams
├── logos/           # OceanLabz branding
├── 3d_models/       # 3D printable enclosure files (STL, STEP)
└── wiring_guides/   # Fritzing / wiring diagrams for examples
```

## Naming Convention

See [Documentation Standards](../docs/contributing/documentation-standards.md#4-image-naming-and-placement) for naming rules.
